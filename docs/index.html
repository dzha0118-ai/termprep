Workflow

一键流水线

从原文到术语准备包，六个步骤全自动完成。

01

文本分析

语言检测、字数统计、领域识别（法律/医学/IT/金融等）、难度评估。

02

术语提取

jieba 分词 • TF-IDF 关键词 • N-gram 词组 • 专有名词识别。

03

关联搜索

有道词典 • Merriam-Webster • Wikipedia — 翻译、释义、搭配、例句。

04

术语入库

自动建立项目术语库，支持状态跟踪、按领域筛选、多库合并。

05

生成报告

综合性的 Markdown 译前准备报告，包含分析、术语、推荐建议。

06

多格式导出

CSV • XLSX (Excel) • TBX (ISO 30042) • JSON — 适配 CAT 工具。

CLI

命令速览

简洁的 CLI 设计，一条命令完成一个任务。

termprep pipeline <file>
全自动流水线
termprep search <term>
搜索关联词汇
termprep analyze <file>
项目分析
termprep extract <file>
术语提取
termprep report <file>
生成译前报告
termprep db init <name>
创建术语库
termprep db list
列出所有词库
termprep export --format xlsx
导出 Excel
